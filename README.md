# 📝 Plagiarism Checker

A lightweight plagiarism detection tool that compares input text against documents or datasets to identify similarity and potential copied content.

---

## 🌟 Features

* Compare text against multiple documents
* Calculate similarity score (percentage)
* Highlight matching content
* Fast and lightweight processing
* Supports multiple input formats (txt)

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* NumPy
* NLTK (optional for preprocessing)
